This website is powered by the following software (in no particular order):
  • Nginx, as webserver and reverse proxy

  • nginx-rtmp-module, as rtmp stream ingest

  • ngx-fancyindex, as cdn directory index

  • Gitea, as git server

  • Photofloat, as image gallery

  • hls.js, as rtmp stream web player

  • Syncthing, as file synchronisation tool

  • Wireguard, as encrypted transport layer

  • BIRD, as internal routing daemon

  • GoTTY, as terminal broadcast tool

  • Mailcow, as mailserver

  • Hugo, as blogging platform

  • Fiche, as pastebin

  • LetsEncrypt, as transport encryption to clients

  • Bitwarden, as password manager

  • HTML5UP, as website designer

  • Debian, as operating system on which to host the above
